Projeto Django com Login via GitHub (OAuth)

OBS: Este projeto é apenas para fins de estudo e aprendizado sobre autenticação com OAuth no Django

Este projeto é uma aplicação Django com autenticação usando **OAuth 2.0 via GitHub**, utilizando a biblioteca `django-allauth`

O objetivo é estudar:
#Autenticação social (GitHub)
#Uso de variáveis de ambiente (`.env`)
#Boas práticas de segurança
#Proteção de rotas com login



Tecnologias usadas

#Python
#Django
#django-allauth
#OAuth 2.0 (GitHub)
#SQLite
#HTML / CSS
